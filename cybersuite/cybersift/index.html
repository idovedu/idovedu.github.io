Usage Guide

How to Use CyberSift

  1. Download the CyberSift tool from here.
  2. Grab sample outputs (e.g., assets.txt or subdomains.txt) and wordlists from the section above.
  3. Install required dependencies: 
    python cybersift.py -i
  4. Run the tool with this command: 
    python cybersift.py -f domain.txt -o recon_results -c -np -t 20
  5. Create a domain.txt file with your target domain (e.g., example.com).
  6. Outputs (e.g., assets.txt, report-example.com.html) are saved in the recon_results folder.
  7. For questions, open an issue on GitHub Issues.
